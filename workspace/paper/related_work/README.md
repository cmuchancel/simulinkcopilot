# Related Work Repository

This folder stores the external PDFs gathered during the literature pass for the Eqn2Sim manuscript.

Files:

- `modelica_1998.pdf`
  - Fritzson and Engelson, "Modelica -- A Language for Equation-Based Physical Modeling and High Performance Simulation"
  - URL: https://modelica.org/papers/1998-06-fritzson-para98LNCS1541-equationbasedmodelingandHighPerformance.pdf
  - Why it matters: early equation-based modeling reference and a baseline for declarative physical-system compilation.

- `openmodelica_mic2020.pdf`
  - Fritzson et al., "The OpenModelica Integrated Environment for Modeling, Simulation, and Model-Based Development"
  - URL: https://www.mic-journal.no/PDF/2020/MIC-2020-4-1.pdf
  - Why it matters: mature open-source equation-based compiler and toolchain baseline.

- `modelingtoolkit_2021.pdf`
  - Ma et al., "ModelingToolkit: A Composable Graph Transformation System for Equation-Based Modeling"
  - URL: https://arxiv.org/pdf/2103.05244.pdf
  - Why it matters: state-of-the-art symbolic transformation and solver-lowering comparison point.

- `modia_2021.pdf`
  - Elmqvist et al., "Modia -- Equation Based Modeling and Domain Specific Algorithms"
  - URL: https://2021.international.conference.modelica.org/proceedings/papers/Modelica2021session1B_paper1.pdf
  - Why it matters: embedded equation-based modeling in Julia with domain-specific lowering.

- `simcheck_2010.pdf`
  - Roy and Shankar, "SimCheck: An Expressive Type System for Simulink"
  - URL: https://www.csl.sri.com/people/shankar/simcheck.pdf
  - Why it matters: analysis and checking of existing Simulink models rather than equation-to-model synthesis.

- `simulink_getting_started_2025.pdf`
  - MathWorks, "Simulink Getting Started Guide"
  - URL: https://www.mathworks.com/help/pdf_doc/simulink/simulink_gs.pdf
  - Why it matters: industrial baseline for manual block-diagram construction.

- `simscape_users_guide_2025b.pdf`
  - MathWorks, "Simscape User's Guide"
  - URL: https://www.mathworks.com/help/pdf_doc/simscape/simscape_ug.pdf
  - Why it matters: industrial equation-oriented modeling baseline inside the MATLAB ecosystem.

Note:
- `matlab_python_cheatsheet_2025.pdf` was excluded from the manuscript evidence set because the retrieved URL resolved to HTML rather than a valid PDF.
